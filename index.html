¿Qué es un título secundario?

En HTML los títulos van por niveles, del h1 al h6.

<h1> sería el título principal, <h2> sería un título secundario

Un título secundario organiza el contenido.

Anatomía del HTML

HTML se compone de elementos, y cada elemento suele tener: Etiqueta de apertura - Contenido - Etiqueta de cierre

Ejemplo de un elemento completo:

<etiqueta atributo="valor">Contenido</etiqueta>

¿Qué es una etiqueta?

Una etiqueta es una instrucción para el navegador. Le dice qué es cada cosa.

Ejemplos:

<p> → párrafo

<h1> → título

<a> → enlace

<img> → imagen

<a> →

¿Qué es un atributo?

Un atributo añade información extra a una etiqueta. Le da contexto o comportamiento.

<a href="https://google.com">Ir a Google</a>

Siempre va entre comillas. El orden y la disciplina importan, incluso en código.

¿Qué diferencia hay entre una “etiqueta de apertura” y una de “apertura y cierre”?

Etiquetas con apertura y cierre

Tienen contenido entre ambas.

Ejemplo:

Este es un párrafo

"p" apertura "/p" cierre (Ver código)

Etiquetas de solamente apertura (Autocontenidas)

No tienen contenido interno

Ejemplo:

<img src="img/foto.jpg">


No cierran porque no hay nada que encerrar. Simple y lógico.

¿Cuáles son las etiquetas más usadas y de qué tipo son?

Estructurales

<html>

<head>

<body>

Texto y contenido

<h1> a <h6> → títulos

<p> → párrafo

<br> → salto de línea

<img> → imagen

<span> → texto en línea

<strong> → texto en negrita, importante

<em> → Enfásis, texto en cursiva

Enlaces

<a> → enlace

<img> → imagen (solo apertura)

Listas

<ul> → lista desordenada

<ol> → lista ordenada

<li> → elemento de lista

Contenedores

<div> → contenedor genérico

<section> → sección del documento con significado

<article> → artículo o contenido independiente
